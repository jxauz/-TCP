# -TCP
用python编译器    写一个TCP转发到串口的客户端程序
pycharm编译器运行代码
打开链接，获取服务器（端口号）
http://tt.ai-thinker.com:8000/ttcloud

可能遇到的问题：No module named 'serial'
解决：1.打开pycharm，然后点击文件（file），再点击（设置）setting
     2.进入setting（设置），找到python解释器
     3.点击 +号 安装，搜索pyserial 安装软件包
